# Components

Reusable components designed for OpenAudioTools devices but can be integrated into other designs.

<div class="grid cards" markdown>

- :material-monitor: __DESPee__

    ESP32-S3 display board with a 4.3" 800×480 capacitive touch LCD and 3 rotary encoders. Connects to any host MCU via a 6-pin UART link and renders widgets using LVGL.

    [:octicons-issue-opened-16: Issue](https://github.com/openaudiotools/about/issues/3)
    · [:octicons-comment-discussion-16: Discussion](https://github.com/openaudiotools/about/discussions/categories/despee)
    · [:octicons-arrow-right-16: Read more](https://github.com/openaudiotools/despee)

- :material-headphones: __PHONEE__

    Reusable headphone output module — TPA6132A2 ground-referenced stereo headphone amp, PCB-mount volume pot, and 1/4" TRS jack on a ~30×20mm 2-layer PCB.

    [:octicons-issue-opened-16: Issue](https://github.com/openaudiotools/about/issues/15)
    · [:octicons-comment-discussion-16: Discussion](https://github.com/openaudiotools/about/discussions/categories/phonee)
    · [:octicons-arrow-right-16: Read more](https://github.com/openaudiotools/phonee)
