# Devices

The OpenAudioTools ecosystem consists of five devices. Some have graduated to their own repositories; others remain at the concept stage.

| Device | Description | Link |
|--------|-------------|------|
| MixTee | Digital audio mixer, recorder, and USB audio interface | [github.com/openaudiotools/mixtee](https://github.com/openaudiotools/mixtee) |
| SynTee | Standalone virtual sound module / synthesizer | [github.com/openaudiotools/syntee](https://github.com/openaudiotools/syntee) |
| HubTee | USB / MIDI / Ethernet hub with clock | [concept page](hubtee.md) |
| VoiceTee | Small microphone preamp module | [concept page](voicetee.md) |
| StringTee | Small guitar/instrument preamp module | [concept page](stringtee.md) |
